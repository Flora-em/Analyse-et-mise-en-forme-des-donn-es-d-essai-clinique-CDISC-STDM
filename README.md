But du projet de l’essai clinique : analyser, interpréter et créer un code afin de pouvoir visualiser les différentes informations de chacune des étapes du schéma : 4 parties 
1. **flowchart** : Calculer des effectifs permettant de compléter le graphique avec les flux de patients selon le modèle proposé
2. **Table 1** : Créer une table descriptive présentant les caractéristiques de la population étudiée à l'inclusion (baseline), stratifiée par bras de traitement
3. **Table 2** :  Élaborer un tableau de résultats présentant :
- Le critère de jugement principal avec analyse statistique complète
- Estimations d'effet (différence de proportions) avec intervalles de confiance à 95%
- Valeurs de p et significativité statistique
4.**Visualisation graphique** : Produire une figure de qualité publication illustrant : L'évolution temporelle du critère principal (comme celle de la figure 1 dans Ling et al)

Compétences :
- analyse des données sous format tabulaire : pandas / polars
- manipulation des données et analyse statistique : numpy / scipy
- construction de graphiques : altair / plotly.express
- analyse et interprétation scientifique des données
